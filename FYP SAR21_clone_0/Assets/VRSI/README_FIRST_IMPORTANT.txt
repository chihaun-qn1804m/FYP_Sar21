If this is the first time you are importing VRSI Unity Package, Please Do The Following Steps Before Try The VRSI Demo In The VRSI/Demo Folder.

1) Open The VRSI_Import_Assist Scene (Located In VRSI/IMPORT_ASSIST Folder)
2) Press Play To Run The Scene
3) Done!

VRSI_Import_Assist Will Re-Import The VRSI Examples Library and Repack The Textures To The VRSI Format.

ENJOY.

(P.S. To Generate Your Own VRSI's Start VRSI Creation Studio Located In Assets/VRSI/VRSI_CREATOR_START_HERE/VRSI_CREATOR )
